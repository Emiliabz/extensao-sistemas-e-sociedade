## 🎯 Sobre o Projeto
Este repositório armazena a solução tecnológica desenvolvida como atividade extensionista para a comunidade local de **Jundiaí - SP**. 

O objetivo principal do projeto foi promover o **letramento de dados (Data Literacy)** e a inclusão digital de microempreendedoras locais (MEIs), ajudando-as a migrar a gestão de seus negócios de anotações puramente manuais (cadernos) para um sistema de informação gerencial simplificado e automatizado.

---

## 💻 A Solução Desenvolvida
Foi estruturado um sistema de controle de fluxo de caixa e gestão de estoque utilizando planilhas automatizadas. A solução foi desenhada com foco em usabilidade e acessibilidade, aplicando conceitos práticos estudados na disciplina:

* **Aba 1 - Registro de Vendas:** Automação do cálculo de faturamento total utilizando fórmulas aritméticas e funções de agregação (`SOMA`).
* **Aba 2 - Controle de Estoque:** Implementação de lógica condicional através da fórmula `=SE()` para gerar alertas automáticos de reposição de mercadorias baseados em estoque mínimo ideal.

Com isso, o projeto demonstra na prática como os Sistemas de Informação podem atuar diretamente como ferramentas de transformação social, reduzindo a vulnerabilidade econômica e aumentando a eficiência de pequenos negócios da comunidade.
